# 11.2. Distribution et incertitude

## Types d'incertitude en apprentissage automatique

### Incertitude épistémique
- **Définition** : manque de connaissances sur le modèle
- **Réductible** : avec plus de données
- **Exemple** : paramètres du modèle incertains

### Incertitude aléatoire
- **Définition** : inhérente au processus de génération des données
- **Irréductible** : même avec données infinies
- **Exemple** : bruit de mesure, variabilité naturelle

## Quantification de l'incertitude

### Approche bayésienne
- **Poids distribués** : p(w|D) distribution postérieure
- **Prédiction** : p(y|x,D) = ∫ p(y|x,w) p(w|D) dw
- **Approximation** : Monte Carlo, variational inference

### Méthodes d'ensemble
- **Multiple modèles** : entraînés différemment
- **Variance** : entre les prédictions des modèles
- **Deep ensembles** : plusieurs réseaux indépendants

### Bootstrap
- **Échantillonnage** : avec remise des données
- **Distribution** : des estimateurs
- **Intervalles** : de confiance non-paramétriques

## Modèles bayésiens

### Réseaux de neurones bayésiens
- **Poids probabilistes** : distributions sur les poids
- **Inférence** : variational inference, MCMC
- **Coût** : computationnellement intensif

### MC Dropout
- **Dropout à l'inférence** : activation du dropout en test
- **Incertitude** : variance des prédictions multiples
- **Approximation** : de l'inférence bayésienne

## Calibration

### Définition
P(ŷ = y | confidence = p) = p

- **Overconfidence** : prédictions trop sûres
- **Underconfidence** : prédictions trop hésitantes
- **Calibration parfaite** : courbe de calibration diagonale

### Méthodes de calibration
- **Temperature scaling** : post-processing simple
- **Platt scaling** : régression logistique
- **Isotonic regression** : non-paramétrique

## Incertitude dans les décisions

### Bandits multi-bras
- **Exploration vs exploitation** : trade-off fondamental
- **UCB** : Upper Confidence Bound
- **Thompson sampling** : échantillonnage bayésien

### Active learning
- **Query strategy** : sélection d'exemples à étiqueter
- **Uncertainty sampling** : exemples les plus incertains
- **Diversity** : couverture de l'espace des entrées

### Safe reinforcement learning
- **Contraintes de sécurité** : bornes sur les actions
- **Risk-sensitive** : aversion au risque
- **Robustesse** : face aux perturbations

# 11.3. Bayésianisme vs fréquentisme

## Deux paradigmes fondamentaux

### Approche fréquentiste
- **Probabilités** : limites de fréquences relatives
- **Paramètres** : valeurs fixes mais inconnues
- **Inférence** : estimation ponctuelle + intervalles de confiance
- **Tests** : rejet de H₀ si p-value < α

### Approche bayésienne
- **Probabilités** : degrés de croyance rationnels
- **Paramètres** : variables aléatoires avec distributions
- **Inférence** : distribution postérieure complète
- **Décisions** : minimisation de la perte espérée

## Comparaison méthodologique

### Avantages fréquentistes
- **Objectivité** : pas de priors subjectifs
- **Répétabilité** : résultats stables entre analystes
- **Simplicité computationnelle** : formules fermées souvent disponibles

### Avantages bayésiens
- **Cohérence logique** : manipulation probabiliste rigoureuse
- **Incorporation du prior** : connaissances expertes intégrées
- **Quantification naturelle** : de l'incertitude
- **Petits échantillons** : robustesse avec peu de données

## Applications pratiques

### A/B testing
- **Fréquentiste** : p-value sur différence de moyennes
- **Bayésien** : P(A > B | données)
- **Séquentiel** : arrêt précoce possible en bayésien

### Modèles hiérarchiques
- **Partage d'information** : entre groupes similaires
- **Shrinkage** : paramètres tirés vers la moyenne globale
- **Naturellement bayésien** : priors sur les hyperparamètres

### Big Data
- **Fréquentiste** : asymptotique valide
- **Bayésien** : computation intensive (MCMC)
- **Approximations** : variational inference pour scalabilité

## Ponts entre paradigmes

### Correspondances asymptotiques
- **Credible intervals** ≈ confidence intervals (grands échantillons)
- **Posterior mode** ≈ MLE
- **Convergence** : vers mêmes conclusions

### Empirical Bayes
- **Prior estimé** : à partir des données elles-mêmes
- **Philosophie fréquentiste** : mais esprit bayésien
- **Applications** : shrinkage, smoothing

## Perspectives modernes

### Réconciliation pragmatique
- **Meilleur outil** : selon le problème
- **Subjectivité assumée** : priors comme choix méthodologiques
- **Computation** : MCMC rend le bayésien pratique

### Futur de l'inférence
- **Auto-tuning priors** : apprentissage automatique des priors
- **Hybrid methods** : combinaison des avantages
- **Débat ouvert** : philosophie de la statistique continue

# 12.4. Applications : deep learning, finance, industrie

## Deep Learning

### Architectures optimisées
- **ResNet** : connexions résiduelles pour réseaux très profonds
- **DenseNet** : connexions denses entre couches
- **EfficientNet** : scaling compound (profondeur, largeur, résolution)

### Optimiseurs spécialisés
- **AdamW** : découplage du weight decay
- **Lookahead** : anticipation du gradient
- **SAM** : Sharpness-Aware Minimization pour paysages lisses

### Compression et accélération
- **Pruning** : élimination de poids non essentiels
- **Quantization** : réduction de précision (INT8, FP16)
- **Knowledge distillation** : transfert vers modèles plus petits

## Finance

### Gestion de portefeuille
- **Markowitz** : frontière efficiente moyenne-variance
- **Black-Litterman** : incorporation de vues d'experts
- **Risk parity** : allocation égale du risque

### Pricing de dérivés
- **Monte Carlo** : simulation de chemins de prix
- **Arbres binomiaux/trinomiaux** : discrétisation du temps
- **PDE** : résolution de Black-Scholes et extensions

### Trading algorithmique
- **Market making** : cotation optimale bid/ask
- **Execution** : minimisation de l'impact de marché
- **Statistical arbitrage** : exploitation d'inefficiences

## Industrie

### Supply chain
- **Inventory optimization** : EOQ, safety stock
- **Production planning** : MRP, JIT
- **Network design** : localisation d'entrepôts

### Manufacturing
- **Scheduling** : ordonnancement de production
- **Quality control** : SPC, acceptance sampling
- **Predictive maintenance** : prévision de pannes

### Énergie
- **Unit commitment** : planification de production électrique
- **Economic dispatch** : répartition optimale de charge
- **Demand forecasting** : prévision de consommation

## Transport et logistique

### Routage
- **TSP** : exact pour petites instances, heuristiques pour grandes
- **VRP** : véhicules multiples avec capacités
- **PDP** : pickup and delivery problems

### Aviation
- **Crew scheduling** : assignation des équipages
- **Fleet assignment** : affectation avions aux routes
- **Revenue management** : tarification dynamique

## Santé

### Essais cliniques
- **Design** : randomisation, stratification
- **Adaptive trials** : modification en cours d'essai
- **Sample size** : calcul de puissance statistique

### Pharmacocinétique
- **Modèles compartimentaux** : ADME
- **Dose optimization** : efficacité vs toxicité
- **Population PK** : modèles mixtes

### Resource allocation
- **Hospital scheduling** : personnel, blocs opératoires
- **Ambulance location** : couverture territoriale
- **Vaccine distribution** : optimisation logistique

## Défis contemporains

### Optimisation sous incertitude
- **Stochastique** : espérance sur scénarios
- **Robust** : pire cas garanti
- **Distributionally robust** : familles de distributions

### Multi-objectif
- **Front de Pareto** : solutions non-dominées
- **Scalarization** : pondération des objectifs
- **NSGA-II** : algorithme évolutionnaire multi-objectif

### Grande échelle
- **Décomposition** : Dantzig-Wolfe, Benders
- **Parallélisation** : multi-cœur, distribué
- **Approximation** : méthodes randomisées

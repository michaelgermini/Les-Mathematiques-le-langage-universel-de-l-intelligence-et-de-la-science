# 12.1. Convexité

## Ensembles convexes

### Définition
Un ensemble C est convexe si ∀x,y ∈ C, ∀λ∈[0,1] : λx + (1-λ)y ∈ C

### Propriétés
- **Intersection** : d'ensembles convexes est convexe
- **Image affine** : par fonction affine est convexe
- **Enveloppe convexe** : plus petit ensemble convexe contenant un ensemble donné

### Exemples importants
- **Simplexe** : {x | Σ x_i = 1, x_i ≥ 0}
- **Boule** : {x | ||x|| ≤ r}
- **Polyèdre** : intersection d'hyperplans et demi-espaces

## Fonctions convexes

### Définition
f convexe si ∀x,y, ∀λ∈[0,1] : f(λx + (1-λ)y) ≤ λf(x) + (1-λ)f(y)

### Caractérisations
- **Inégalité de Jensen** : E[f(X)] ≥ f(E[X])
- **Condition du premier ordre** : f(y) ≥ f(x) + ∇f(x)·(y-x)
- **Condition du second ordre** : H ≽ 0 (hessienne semi-définie positive)

### Classes importantes
- **Affine** : f(x) = a·x + b
- **Quadratique** : (1/2)x^T A x + b·x + c, avec A ≽ 0
- **Exponentielle** : e^{a·x}, normes L_p (p ≥ 1)

## Optimisation convexe

### Problème standard
min f(x) sous contraintes g_i(x) ≤ 0, h_j(x) = 0

- **Objectif** : fonction convexe f
- **Contraintes** : g_i convexes, h_j affines

### Conditions KKT (Karush-Kuhn-Tucker)
- **Stationnarité** : ∇L(x,λ,μ) = 0
- **Faisabilité primale** : g_i(x) ≤ 0, h_j(x) = 0
- **Faisabilité duale** : λ_i ≥ 0
- **Complémentarité** : λ_i g_i(x) = 0

### Dualité
- **Lagrangien** : L(x,λ) = f(x) + Σ λ_i g_i(x)
- **Problème dual** : max_λ min_x L(x,λ)
- **Écart de dualité** : borne supérieure sur l'optimum primal

## Algorithmes d'optimisation convexe

### Gradient projeté
x_{k+1} = P_C [x_k - α ∇f(x_k)]

- **Projection** : sur l'ensemble convexe C
- **Convergence** : O(1/k) pour fonctions convexes
- **Applications** : contraintes simples (boîtes, simplexe)

### Méthodes de point intérieur
- **Barrière logarithmique** : -t Σ log(-g_i(x))
- **Central path** : suite de solutions approchées
- **Primal-dual** : résolution simultanée primal et dual

### Programmation quadratique (QP)
min (1/2)x^T Q x + c^T x sous A x ≤ b

- **Solution** : méthodes active set
- **Complexité** : polynomiale
- **Applications** : SVM, optimisation de portefeuille

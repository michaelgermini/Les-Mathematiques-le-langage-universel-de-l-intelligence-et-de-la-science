# 14.2. Génétique, statistique et IA médicale

## Introduction

La révolution génomique a généré des quantités massives de données biologiques. Les mathématiques — statistique, apprentissage automatique, bioinformatique — sont indispensables pour extraire le sens de ces données et les traduire en avancées médicales. Ce chapitre explore l'intersection de la génétique, de la statistique et de l'intelligence artificielle en médecine.

## 1. Génétique quantitative

### 1.1 Héritabilité

#### Décomposition de la variance phénotypique
```
V_P = V_G + V_E
```
où :
- V_P : variance phénotypique totale
- V_G : variance génétique
- V_E : variance environnementale

#### Héritabilité au sens large
```
H² = V_G / V_P
```
Proportion de la variance expliquée par les facteurs génétiques.

#### Héritabilité au sens étroit
```
h² = V_A / V_P
```
où V_A est la variance additive (effets additifs des allèles).

#### Estimation
- **Études de jumeaux** : comparer MZ (identiques) et DZ (fraternels)
- **Études familiales** : corrélations parent-enfant
- **GCTA** : estimation à partir de données génomiques (SNPs)

### 1.2 Études d'association pangénomiques (GWAS)

#### Principe
Tester l'association entre chaque variant génétique (SNP) et un trait/maladie.

#### Modèle de régression
```
y = β₀ + β₁·SNP + covariables + ε
```
où SNP ∈ {0, 1, 2} (nombre d'allèles mineurs).

#### Correction pour tests multiples
Avec ~1 million de SNPs testés :
- **Bonferroni** : p < 5×10⁻⁸ (seuil génome-wide)
- **FDR** : contrôle du taux de faux positifs

#### Manhattan plot
```
-log₁₀(p)
    │    ●
    │   ●●●  ●
    │  ●●●●●●●●●    ●
    │ ●●●●●●●●●●●●●●●●●  ●
    │●●●●●●●●●●●●●●●●●●●●●●●
    └─────────────────────────→ Position chromosomique
         Chr1   Chr2  ...  Chr22
```

#### Résultats typiques
- Maladies complexes : centaines de variants de petit effet
- Maladies mendéliennes : un ou quelques variants de grand effet

### 1.3 Scores de risque polygénique (PRS)

#### Définition
```
PRS = Σⱼ βⱼ × SNPⱼ
```
Somme pondérée des effets de nombreux variants.

#### Construction
1. Estimer les βⱼ dans une cohorte de découverte (GWAS)
2. Sélectionner les SNPs (seuil de p-value, clumping)
3. Calculer le score dans une cohorte indépendante

#### Applications
- Stratification du risque de maladie
- Médecine préventive personnalisée
- Prédiction de réponse aux traitements

#### Limitations
- Transférabilité limitée entre populations
- Explique une fraction modeste de l'héritabilité
- Questions éthiques (discrimination, déterminisme)

## 2. Statistique en recherche biomédicale

### 2.1 Analyse de survie

#### Données censurées
Les temps de survie sont souvent incomplets :
- **Censure à droite** : le patient est encore vivant à la fin de l'étude
- **Censure à gauche** : l'événement s'est produit avant l'observation

#### Estimateur de Kaplan-Meier
```
Ŝ(t) = ∏_{tᵢ≤t} (1 - dᵢ/nᵢ)
```
où dᵢ = décès au temps tᵢ, nᵢ = à risque juste avant tᵢ.

#### Test du log-rank
Compare deux courbes de survie :
```
χ² = (O₁ - E₁)² / E₁ + (O₂ - E₂)² / E₂
```
sous H₀ : les deux groupes ont la même survie.

#### Modèle de Cox (risques proportionnels)
```
h(t|X) = h₀(t) × exp(β'X)
```
- h₀(t) : risque de base (non paramétrique)
- exp(βⱼ) : hazard ratio pour la variable j

### 2.2 Modèles mixtes

#### Motivation
Données hiérarchiques : patients dans des hôpitaux, mesures répétées sur un individu.

#### Modèle linéaire mixte
```
y = Xβ + Zu + ε
```
où :
- β : effets fixes (population)
- u : effets aléatoires (individus/groupes)
- Z : matrice de design pour les effets aléatoires

#### Estimation
- **REML** : Restricted Maximum Likelihood
- **BLUP** : Best Linear Unbiased Prediction pour u

### 2.3 Analyse de données omiques

#### Types de données
| Omique | Mesure | Dimension typique |
|--------|--------|-------------------|
| **Génomique** | Variants (SNPs) | ~10⁶ |
| **Transcriptomique** | Expression des gènes | ~20 000 |
| **Protéomique** | Abondance des protéines | ~10 000 |
| **Métabolomique** | Métabolites | ~1 000 |
| **Épigénomique** | Méthylation ADN | ~500 000 |

#### Défis statistiques
- **Haute dimension** : p >> n
- **Corrélations** : gènes co-régulés
- **Bruit** : mesures techniques
- **Intégration** : combiner plusieurs omiques

#### Méthodes
- Régularisation (Lasso, Elastic Net)
- Réduction de dimension (PCA, t-SNE, UMAP)
- Analyse d'enrichissement (Gene Ontology, pathways)
- Réseaux de co-expression

## 3. Intelligence artificielle en médecine

### 3.1 Diagnostic par imagerie

#### Applications

| Domaine | Tâche | Performance |
|---------|-------|-------------|
| **Dermatologie** | Classification des lésions cutanées | Niveau dermatologue |
| **Ophtalmologie** | Détection de rétinopathie diabétique | Approuvé FDA |
| **Radiologie** | Détection de nodules pulmonaires | Aide au diagnostic |
| **Pathologie** | Analyse de lames histologiques | En développement |
| **Cardiologie** | Analyse d'ECG | Détection d'arythmies |

#### Architectures
- **CNN** : classification d'images
- **U-Net** : segmentation
- **Vision Transformers** : état de l'art récent

#### Défis
- **Données annotées** : coûteuses en médecine
- **Généralisation** : entre hôpitaux, populations
- **Interprétabilité** : comprendre les décisions
- **Régulation** : certification des dispositifs médicaux

### 3.2 Découverte de médicaments

#### Pipeline traditionnel
```
Cible → Hit → Lead → Préclinique → Clinique → Approbation
        ↑
    ~10-15 ans, ~2 milliards $
```

#### Applications de l'IA

| Étape | Application IA |
|-------|----------------|
| **Identification de cibles** | Analyse de réseaux biologiques |
| **Criblage virtuel** | Prédiction d'affinité ligand-cible |
| **Optimisation de leads** | Génération de molécules |
| **Prédiction ADMET** | Absorption, toxicité |
| **Repositionnement** | Nouvelles indications pour médicaments existants |

#### AlphaFold et la prédiction de structure
- Prédit la structure 3D des protéines à partir de la séquence
- Révolution pour la conception de médicaments
- Accélère la compréhension des mécanismes

### 3.3 Médecine personnalisée

#### Principe
Adapter le traitement au profil individuel du patient.

#### Sources de données
- Génome (pharmacogénomique)
- Transcriptome tumoral
- Historique médical (dossiers électroniques)
- Données de capteurs (wearables)

#### Applications
- **Oncologie** : thérapies ciblées selon les mutations
- **Pharmacogénomique** : dosage selon le métabolisme
- **Prédiction de risque** : scores polygéniques

### 3.4 Traitement du langage naturel médical

#### Applications
- Extraction d'information des dossiers médicaux
- Aide à la rédaction de comptes-rendus
- Chatbots de triage
- Recherche bibliographique automatisée

#### Défis spécifiques
- Vocabulaire médical spécialisé
- Abréviations et jargon
- Confidentialité des données
- Conséquences des erreurs

## 4. Considérations éthiques et pratiques

### 4.1 Biais et équité

#### Sources de biais
- **Données d'entraînement** : sous-représentation de certaines populations
- **Annotation** : subjectivité des experts
- **Algorithme** : optimisation pour la moyenne

#### Conséquences
- Performance dégradée pour certains groupes
- Amplification des inégalités de santé
- Perte de confiance

#### Mitigation
- Données diversifiées
- Audits de performance par sous-groupe
- Fairness constraints dans l'optimisation

### 4.2 Interprétabilité

#### Importance en médecine
- Confiance des cliniciens
- Responsabilité légale
- Découverte scientifique

#### Méthodes
- **SHAP** : importance des features
- **Grad-CAM** : régions d'attention (images)
- **Modèles intrinsèquement interprétables** : arbres, règles

### 4.3 Validation clinique

#### Niveaux de preuve
1. **Rétrospectif** : données historiques
2. **Prospectif** : nouvelle collecte, même site
3. **Externe** : validation multi-sites
4. **Essai clinique** : impact sur les outcomes patients

#### Régulation
- **FDA** : Software as a Medical Device (SaMD)
- **CE** : marquage européen
- **Surveillance post-marché** : monitoring continu

### 4.4 Confidentialité

#### Défis
- Données de santé sensibles
- Ré-identification possible (génomique)
- Partage pour la recherche

#### Solutions
- **Anonymisation** : suppression des identifiants
- **Differential privacy** : bruit ajouté
- **Federated learning** : entraînement sans centraliser les données
- **Environnements sécurisés** : accès contrôlé

---

## Conclusion

L'intersection de la génétique, de la statistique et de l'IA transforme la médecine :

1. **Génétique** : comprendre les bases moléculaires des maladies
2. **Statistique** : analyser rigoureusement les données biomédicales
3. **IA** : automatiser le diagnostic, accélérer la découverte

Ces avancées promettent une médecine plus précise et personnalisée, mais soulèvent des défis éthiques (équité, confidentialité, responsabilité) qui nécessitent une attention constante.

# 13.1. Modélisation économique

## Introduction

L'économie mathématique utilise des outils formels pour modéliser les comportements des agents économiques et les mécanismes de marché. Ce chapitre présente les fondements mathématiques de la théorie économique moderne, de la microéconomie des choix individuels à la macroéconomie de la croissance.

---

## 1. Théorie des choix rationnels

### 1.1 Préférences et utilité

#### Axiomes des préférences
Une relation de préférence ≿ sur un ensemble X de choix est **rationnelle** si :

| Axiome | Définition | Interprétation |
|--------|------------|----------------|
| **Complétude** | ∀x,y : x ≿ y ou y ≿ x | Tout est comparable |
| **Transitivité** | x ≿ y et y ≿ z ⟹ x ≿ z | Cohérence |
| **Continuité** | {y : y ≿ x} et {y : x ≿ y} fermés | Pas de sauts |

#### Fonction d'utilité
**Théorème** (Debreu) : Si ≿ satisfait ces axiomes sur un ensemble connexe, il existe une fonction continue U : X → ℝ telle que :
```
x ≿ y ⟺ U(x) ≥ U(y)
```

U est unique à transformation monotone croissante près.

#### Utilité cardinale vs ordinale
- **Ordinale** : seul l'ordre compte (U équivalent à f(U) pour f croissante)
- **Cardinale** : les différences ont un sens (décisions sous risque)

### 1.2 Maximisation de l'utilité

#### Problème du consommateur
```
max U(x₁, x₂, ..., xₙ)
sous contrainte : p·x ≤ m
```
où p est le vecteur des prix et m le revenu.

#### Conditions du premier ordre (Lagrangien)
```
∂U/∂xᵢ = λ pᵢ pour tout i
```
où λ est le multiplicateur de Lagrange (utilité marginale du revenu).

#### Taux marginal de substitution
```
TMS = (∂U/∂x₁)/(∂U/∂x₂) = p₁/p₂
```
À l'optimum, le TMS égale le rapport des prix.

### 1.3 Demande et élasticité

#### Fonction de demande
```
xᵢ*(p, m) = argmax U(x) sous p·x ≤ m
```

#### Élasticités

| Type | Formule | Interprétation |
|------|---------|----------------|
| **Prix propre** | εᵢᵢ = (∂xᵢ/∂pᵢ)(pᵢ/xᵢ) | Sensibilité au prix |
| **Prix croisée** | εᵢⱼ = (∂xᵢ/∂pⱼ)(pⱼ/xᵢ) | Substituts/compléments |
| **Revenu** | ηᵢ = (∂xᵢ/∂m)(m/xᵢ) | Bien normal/inférieur |

#### Équation de Slutsky
```
∂xᵢ/∂pⱼ = ∂hᵢ/∂pⱼ - xⱼ ∂xᵢ/∂m
```
Décompose l'effet prix en effet substitution et effet revenu.

## 2. Équilibre de marché

### 2.1 Offre et demande

#### Équilibre partiel
```
Qᵈ(p) = Qˢ(p)
```
Le prix d'équilibre p* égalise offre et demande.

#### Stabilité
L'équilibre est stable si :
- Excès de demande → prix augmente
- Excès d'offre → prix diminue

Condition : dQᵈ/dp < dQˢ/dp

### 2.2 Équilibre général (Arrow-Debreu)

#### Cadre
- L biens, I consommateurs, J producteurs
- Dotations initiales ωᵢ pour chaque consommateur
- Technologies de production Yⱼ

#### Définition d'un équilibre
Un équilibre concurrentiel est un vecteur de prix p* et des allocations (x*, y*) tels que :
1. **Maximisation des consommateurs** : xᵢ* maximise Uᵢ sous contrainte budgétaire
2. **Maximisation des producteurs** : yⱼ* maximise p*·y sur Yⱼ
3. **Équilibre des marchés** : Σᵢ xᵢ* = Σᵢ ωᵢ + Σⱼ yⱼ*

#### Théorèmes du bien-être

**Premier théorème** : Tout équilibre concurrentiel est Pareto-optimal.

**Second théorème** : Toute allocation Pareto-optimale peut être atteinte comme équilibre concurrentiel après redistribution des dotations.

#### Existence (Théorème d'Arrow-Debreu)
Sous conditions de convexité et continuité, un équilibre existe.
Démonstration : théorème du point fixe de Kakutani.

### 2.3 Échecs de marché

| Type | Cause | Solution |
|------|-------|----------|
| **Externalités** | Effets non compensés | Taxe pigouvienne, droits de propriété |
| **Biens publics** | Non-rivalité, non-exclusion | Provision publique |
| **Asymétrie d'information** | Sélection adverse, aléa moral | Signaling, screening |
| **Pouvoir de marché** | Monopole, oligopole | Régulation, antitrust |

## 3. Concurrence imparfaite

### 3.1 Monopole

#### Maximisation du profit
```
max π = p(Q)·Q - C(Q)
```

#### Condition du premier ordre
```
MR = MC
p(1 - 1/|ε|) = MC
```
où ε est l'élasticité-prix de la demande.

#### Perte sèche (deadweight loss)
Le monopole produit moins et vend plus cher qu'en concurrence parfaite.

### 3.2 Oligopole

#### Modèle de Cournot (quantités)
Chaque firme choisit sa quantité en prenant celles des autres comme données.

**Équilibre de Nash** :
```
∂πᵢ/∂qᵢ = 0 pour tout i
```

Pour n firmes identiques avec demande linéaire P = a - bQ et coût c :
```
qᵢ* = (a - c) / (b(n+1))
```

#### Modèle de Bertrand (prix)
Les firmes choisissent les prix.
Avec produits homogènes : équilibre = concurrence parfaite (paradoxe de Bertrand).

#### Modèle de Stackelberg (leader-follower)
Le leader anticipe la réaction du suiveur.
Avantage au premier entrant.

### 3.3 Concurrence monopolistique

- Produits différenciés
- Libre entrée et sortie
- Profit nul à long terme
- Variété des produits

## 4. Croissance économique

### 4.1 Modèle de Solow

#### Équations
```
Y = F(K, L) = Kᵅ L¹⁻ᵅ    (production Cobb-Douglas)
K̇ = sY - δK              (accumulation du capital)
L̇ = nL                   (croissance de la population)
```

#### En termes de capital par travailleur k = K/L
```
k̇ = sf(k) - (n + δ)k
```

#### État stationnaire
```
sf(k*) = (n + δ)k*
k* = (s/(n+δ))^(1/(1-α))
```

#### Propriétés
- **Convergence** : les pays pauvres rattrapent les riches
- **Croissance de long terme** : déterminée par le progrès technique (exogène)
- **Règle d'or** : s = α maximise la consommation par tête

### 4.2 Croissance endogène

#### Modèle AK
```
Y = AK
```
Pas de rendements décroissants → croissance perpétuelle.

#### Modèle de Romer (1990)
- Le progrès technique est endogène (R&D)
- Externalités de connaissance
- Rôle du capital humain

#### Modèle de Lucas (1988)
- Accumulation de capital humain
- Effets d'apprentissage

### 4.3 Modèles DSGE

**Dynamic Stochastic General Equilibrium** :

#### Structure
- Agents optimisateurs (ménages, firmes)
- Équilibre général intertemporel
- Chocs stochastiques (productivité, politique monétaire)

#### Équations typiques
```
Euler (consommation) : 1 = β E[R(1+r)]
Phillips (inflation) : π = β E[π'] + κ(y - y*)
Taylor (politique) : i = r* + φ_π π + φ_y (y - y*)
```

#### Applications
- Analyse de politique monétaire
- Effets des chocs
- Prévision macroéconomique

## 5. Économétrie structurelle

### 5.1 Estimation des modèles

#### Maximum de vraisemblance
```
θ̂ = argmax Σᵢ log f(yᵢ|xᵢ; θ)
```

#### Méthode des moments généralisée (GMM)
```
θ̂ = argmin g(θ)' W g(θ)
```
où g(θ) sont les conditions de moments.

### 5.2 Identification

#### Problème
Les paramètres structurels sont-ils identifiables à partir des données ?

#### Instruments
Variables corrélées avec les régresseurs endogènes mais pas avec les erreurs.

#### Expériences naturelles
Exploiter des variations exogènes (réformes, événements).

### 5.3 Causalité

#### Différence-en-différences
```
τ = (Y_traité,après - Y_traité,avant) - (Y_contrôle,après - Y_contrôle,avant)
```

#### Régression sur discontinuité
Exploiter un seuil arbitraire d'attribution du traitement.

## Conclusion

La modélisation économique mathématique permet de :

1. **Formaliser** les comportements rationnels (utilité, profit)
2. **Analyser** les équilibres de marché et leur efficacité
3. **Comprendre** les mécanismes de croissance et leurs déterminants
4. **Évaluer** les politiques économiques de manière rigoureuse

Ces outils, de l'optimisation à la théorie des jeux en passant par les équations différentielles, font de l'économie une science profondément mathématique.

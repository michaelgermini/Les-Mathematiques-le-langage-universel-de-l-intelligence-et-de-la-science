# 13.3. Jeux, stratégies et décisions

## Introduction

La théorie des jeux étudie les interactions stratégiques entre agents rationnels. Fondée par von Neumann et Morgenstern (1944) et développée par Nash, elle s'applique à l'économie, la politique, la biologie et l'informatique. Ce chapitre présente les concepts fondamentaux et leurs applications.

---

## 1. Jeux sous forme normale

### 1.1 Définitions

#### Éléments d'un jeu
- **Joueurs** : N = {1, 2, ..., n}
- **Stratégies** : Sᵢ pour chaque joueur i
- **Payoffs** : uᵢ(s₁, ..., sₙ) pour chaque profil de stratégies

#### Représentation matricielle (2 joueurs)
```
                Joueur 2
              Gauche  Droite
Joueur 1  Haut  (3,3)   (0,5)
          Bas   (5,0)   (1,1)
```
Le premier nombre est le payoff du joueur 1, le second celui du joueur 2.

### 1.2 Concepts de solution

#### Stratégies dominantes
Une stratégie sᵢ **domine strictement** sᵢ' si :
```
uᵢ(sᵢ, s₋ᵢ) > uᵢ(sᵢ', s₋ᵢ) pour tout s₋ᵢ
```

**Élimination itérée** : supprimer les stratégies dominées jusqu'à convergence.

#### Équilibre de Nash
Un profil s* est un **équilibre de Nash** si aucun joueur n'a intérêt à dévier unilatéralement :
```
uᵢ(sᵢ*, s*₋ᵢ) ≥ uᵢ(sᵢ, s*₋ᵢ) pour tout sᵢ et tout i
```

**Théorème de Nash** : Tout jeu fini admet au moins un équilibre de Nash (possiblement en stratégies mixtes).

#### Stratégies mixtes
Une stratégie mixte est une distribution de probabilité sur les stratégies pures :
```
σᵢ ∈ Δ(Sᵢ)
```

L'utilité espérée :
```
uᵢ(σ) = Σₛ (∏ⱼ σⱼ(sⱼ)) uᵢ(s)
```

### 1.3 Jeux classiques

#### Dilemme du prisonnier
```
              Coopérer  Trahir
Coopérer       (3,3)    (0,5)
Trahir         (5,0)    (1,1)  ← Équilibre de Nash
```
L'équilibre (Trahir, Trahir) est Pareto-dominé par (Coopérer, Coopérer).

#### Bataille des sexes
```
              Opéra    Football
Opéra         (3,2)     (0,0)
Football      (0,0)     (2,3)
```
Deux équilibres en stratégies pures, un en mixtes.

#### Jeu de coordination
```
              A         B
A            (2,2)     (0,0)
B            (0,0)     (1,1)
```
Deux équilibres : (A,A) Pareto-domine (B,B).

#### Matching Pennies (jeu à somme nulle)
```
              Pile     Face
Pile         (1,-1)   (-1,1)
Face        (-1,1)    (1,-1)
```
Unique équilibre en stratégies mixtes : (½, ½) pour chaque joueur.

## 2. Jeux dynamiques

### 2.1 Jeux sous forme extensive

#### Représentation par arbre
```
        1
       / \
      L   R
     /     \
    2       2
   / \     / \
  l   r   l   r
(3,1)(0,0)(0,0)(1,3)
```

#### Information parfaite vs imparfaite
- **Parfaite** : chaque joueur connaît l'historique complet
- **Imparfaite** : ensembles d'information (nœuds indistinguables)

### 2.2 Induction à rebours (Backward Induction)

#### Principe
Résoudre l'arbre de la fin vers le début :
1. Aux nœuds terminaux, lire les payoffs
2. À chaque nœud de décision, choisir l'action optimale
3. Remonter jusqu'à la racine

#### Équilibre parfait en sous-jeux (SPE)
Un profil de stratégies est un SPE si il induit un équilibre de Nash dans chaque sous-jeu.

### 2.3 Jeux répétés

#### Jeu répété infiniment
Le jeu de base est joué à chaque période t = 0, 1, 2, ...

#### Facteur d'escompte
```
Utilité totale = Σₜ δᵗ uᵢ(sᵗ)
```
où δ ∈ (0,1) est le facteur d'escompte.

#### Théorème du folk
Si δ est suffisamment proche de 1, tout payoff individuellement rationnel peut être soutenu comme équilibre.

#### Stratégies de représailles
- **Grim trigger** : coopérer jusqu'à la première trahison, puis punir pour toujours
- **Tit-for-tat** : répliquer l'action précédente de l'adversaire

## 3. Théorie des décisions

### 3.1 Décision sous incertitude

#### Cadre
- Actions : A = {a₁, ..., aₘ}
- États du monde : Ω = {ω₁, ..., ωₙ}
- Conséquences : c(a, ω)
- Probabilités : P(ω) ou inconnues

### 3.2 Critères de décision

| Critère | Formule | Attitude |
|---------|---------|----------|
| **Espérance d'utilité** | max_a Σ_ω P(ω) U(c(a,ω)) | Neutre au risque |
| **Maximin** | max_a min_ω U(c(a,ω)) | Pessimiste |
| **Maximax** | max_a max_ω U(c(a,ω)) | Optimiste |
| **Hurwicz** | max_a [α max_ω + (1-α) min_ω] | Compromis |
| **Minimax regret** | min_a max_ω [max_a' U(c(a',ω)) - U(c(a,ω))] | Regret minimal |

### 3.3 Utilité espérée (von Neumann-Morgenstern)

#### Axiomes
1. **Complétude** : toute paire de loteries est comparable
2. **Transitivité** : cohérence des préférences
3. **Continuité** : pas de sauts
4. **Indépendance** : préférences invariantes par mélange

#### Théorème
Sous ces axiomes, il existe U : X → ℝ telle que :
```
L ≿ L' ⟺ E_L[U] ≥ E_L'[U]
```

### 3.4 Aversion au risque

#### Définition
Un agent est **averse au risque** si :
```
U(E[X]) > E[U(X)]
```
pour toute loterie non dégénérée X.

Équivalent : U est concave.

#### Coefficient d'Arrow-Pratt
- **Absolu** : A(w) = -U''(w)/U'(w)
- **Relatif** : R(w) = -w·U''(w)/U'(w)

#### Fonctions d'utilité courantes

| Fonction | U(w) | Aversion |
|----------|------|----------|
| **CARA** | -e^{-αw} | A(w) = α (constante) |
| **CRRA** | w^{1-γ}/(1-γ) | R(w) = γ (constante) |
| **Logarithmique** | ln(w) | R(w) = 1 |

### 3.5 Paradoxes et alternatives

#### Paradoxe d'Allais
Les choix observés violent l'axiome d'indépendance.

#### Prospect Theory (Kahneman-Tversky)
- **Référence** : gains et pertes par rapport à un point de référence
- **Aversion aux pertes** : les pertes pèsent plus que les gains
- **Pondération des probabilités** : surestimation des petites probabilités

## 4. Applications économiques

### 4.1 Enchères

#### Types d'enchères

| Type | Mécanisme | Stratégie optimale |
|------|-----------|-------------------|
| **Anglaise** | Prix croissant, dernier enchérisseur gagne | Enchérir jusqu'à sa valeur |
| **Hollandaise** | Prix décroissant, premier à accepter gagne | Stratégique |
| **Premier prix scellé** | Plus haute offre gagne, paie son offre | Shading (offrir moins que sa valeur) |
| **Second prix (Vickrey)** | Plus haute offre gagne, paie la 2ème | Offrir sa vraie valeur |

#### Théorème d'équivalence des revenus
Sous certaines conditions, toutes les enchères génèrent le même revenu espéré pour le vendeur.

### 4.2 Mécanismes d'incitation

#### Problème principal-agent
- Le **principal** (employeur) veut inciter l'**agent** (employé) à l'effort
- Information asymétrique : l'effort n'est pas observable

#### Contrat optimal
```
w(y) = α + β·y
```
où y est le résultat observable.

Compromis : incitation (β élevé) vs assurance (β faible).

### 4.3 Négociation

#### Solution de Nash
Maximise le produit des gains par rapport au point de désaccord :
```
max (u₁ - d₁)(u₂ - d₂)
```

#### Propriétés
- Efficacité Pareto
- Symétrie
- Invariance aux transformations affines
- Indépendance des alternatives non pertinentes

## 5. Jeux et informatique

### 5.1 Complexité du calcul des équilibres

| Problème | Complexité |
|----------|------------|
| Vérifier un équilibre de Nash | P |
| Trouver un équilibre de Nash | PPAD-complet |
| Équilibre en stratégies dominantes | P |
| Équilibre corrélé | P |

### 5.2 Algorithmes

#### Lemke-Howson
Algorithme de pivot pour les jeux bimatriciels.

#### Support Enumeration
Énumère les supports possibles des stratégies mixtes.

#### Fictitious Play
Chaque joueur joue la meilleure réponse à l'historique des autres.

### 5.3 Apprentissage multi-agents

#### Regret matching
Converge vers un équilibre corrélé.

#### CFR (Counterfactual Regret Minimization)
État de l'art pour les jeux imparfaits (poker).

## 6. Théorie des jeux évolutionnaire

### 6.1 Dynamique de réplication

```
ẋᵢ = xᵢ(fᵢ(x) - f̄(x))
```
où xᵢ est la proportion de la stratégie i et fᵢ son fitness.

### 6.2 Stratégie évolutivement stable (ESS)

Une stratégie s* est ESS si :
1. u(s*, s*) ≥ u(s, s*) pour tout s
2. Si égalité : u(s*, s) > u(s, s)

### 6.3 Applications en biologie

- Conflits animal (hawk-dove)
- Coopération (réciprocité)
- Signaux et communication

## Conclusion

La théorie des jeux et des décisions fournit un cadre rigoureux pour analyser :

1. **Les interactions stratégiques** (équilibre de Nash, jeux répétés)
2. **Les choix sous incertitude** (utilité espérée, aversion au risque)
3. **Les mécanismes économiques** (enchères, contrats, négociation)
4. **L'évolution des comportements** (ESS, dynamiques de réplication)

Ces outils mathématiques éclairent des phénomènes aussi divers que la concurrence entre entreprises, les négociations internationales, l'évolution biologique et la conception d'algorithmes d'IA.

# 5.2. Équations différentielles

## Introduction : Modèles du changement

Les équations différentielles décrivent comment les systèmes évoluent au cours du temps. Elles constituent le langage mathématique privilégié pour modéliser les phénomènes dynamiques dans tous les domaines de la science : physique, biologie, économie, ingénierie.

## Équations différentielles ordinaires (EDO)

### Définition et classification

Une EDO lie une fonction inconnue y(t) et ses dérivées :
F(t, y, y', y'', ..., y^{(n)}) = 0

#### Ordre de l'équation
- **Premier ordre** : y' = f(t,y)
- **Deuxième ordre** : y'' = f(t,y,y')
- **Ordre n** : implique la dérivée n-ième

#### Linéarité
- **Linéaire** : y'' + p(t)y' + q(t)y = r(t)
- **Non-linéaire** : termes en y², y y', etc.

### Solutions et conditions initiales

#### Problème de Cauchy
Pour y' = f(t,y), avec y(t₀) = y₀

- **Existence** : théorème de Picard-Lindelöf (cas lipschitzien)
- **Unicité** : solution unique localement
- **Prolongement** : solution maximale

#### Conditions aux limites
Pour équations d'ordre supérieur :
- y(a) = α, y(b) = β (Dirichlet)
- y'(a) = α, y'(b) = β (Neumann)

## Méthodes de résolution

### Équations à variables séparables

Forme : dy/dx = f(x) g(y)
- **Séparation** : g(y) dy = f(x) dx
- **Intégration** : ∫ g(y) dy = ∫ f(x) dx + C
- **Solution implicite**

Exemple : dy/dx = x/y
y dy = x dx ⇒ (1/2)y² = (1/2)x² + C ⇒ x² - y² = K

### Équations linéaires du premier ordre

Forme : y' + p(t)y = q(t)
- **Facteur intégrant** : μ(t) = e^{∫p(t) dt}
- **Solution** : y = [∫ μ q dt + C] / μ

### Équations de Bernoulli

y' + p(t)y = q(t) y^n
- **Changement de variable** : z = y^{1-n}
- **Réduction** : équation linéaire en z

### Systèmes d'équations

dX/dt = A X + B(t)
- **Solution matricielle** : X(t) = e^{At} X₀ + ∫ e^{A(t-s)} B(s) ds
- **Méthode d'Euler** : approximation numérique

## Équations linéaires d'ordre supérieur

### Équation homogène

y'' + p(t)y' + q(t)y = 0
- **Solution générale** : combinaison linéaire de 2 solutions indépendantes
- **Wronskien** : critère d'indépendance

### Équation complète

y'' + p y' + q y = r(t)
- **Solution particulière** : méthode de variation des constantes
- **Solution générale** : homogène + particulière

### Équations à coefficients constants

y'' + a y' + b y = r(t)
- **Équation caractéristique** : r² + a r + b = 0
- **Racines** : déterminent la forme de la solution

## Méthodes numériques

### Méthode d'Euler

y_{n+1} = y_n + h f(t_n, y_n)
- **Pas h** : précision vs stabilité
- **Erreur locale** : O(h²)
- **Erreur globale** : O(h)

### Méthode de Runge-Kutta d'ordre 4

Approximation plus précise :
k₁ = h f(t_n, y_n)
k₂ = h f(t_n + h/2, y_n + k₁/2)
k₃ = h f(t_n + h/2, y_n + k₂/2)
k₄ = h f(t_n + h, y_n + k₃)
y_{n+1} = y_n + (k₁ + 2k₂ + 2k₃ + k₄)/6

### Méthodes multi-étapes

- **Adams-Bashforth** : extrapolation
- **Adams-Moulton** : interpolation
- **Prédicteur-correcteur**

## Équations différentielles partielles (EDP)

### Classification

#### EDP d'ordre 1
Forme : a u_x + b u_y = c
- **Méthode des caractéristiques**
- **Solutions** : le long des courbes caractéristiques

#### EDP d'ordre 2
a u_xx + 2h u_xy + b u_yy + ... = 0

- **Elliptique** : ab - h² > 0 (équilibre)
- **Parabolique** : ab - h² = 0 (diffusion)
- **Hyperbolique** : ab - h² < 0 (ondes)

### Équations célèbres

#### Équation de Laplace
Δu = 0 (équilibre, potentiel électrique)

#### Équation de la chaleur
∂u/∂t = k Δu (diffusion thermique)

#### Équation des ondes
∂²u/∂t² = c² Δu (propagation)

## Applications en sciences

### Physique

#### Mécanique newtonienne
- **Oscillateur harmonique** : m ÿ + k y = 0
- **Pendule** : ÿ + (g/l) sin y = 0
- **Chute libre** : ÿ = -g

#### Électromagnétisme
- **Circuit RL** : L di/dt + R i = V
- **Ondes électromagnétiques** : équations de Maxwell

### Biologie

#### Croissance démographique
dn/dt = r n (modèle de Malthus)

#### Épidémies (SIR)
dS/dt = -β S I
dI/dt = β S I - γ I
dR/dt = γ I

#### Cinétique enzymatique
d[S]/dt = -k [E][S] (Michaelis-Menten)

### Économie

#### Croissance économique
dY/dt = s Y (modèle de Solow)

#### Modèles de prédateur-proie
dx/dt = α x - β x y
dy/dt = -γ y + δ x y (Lotka-Volterra)

### Chimie

#### Cinétique de réaction
d[A]/dt = -k [A] (réaction d'ordre 1)

#### Diffusion
∂c/∂t = D Δc (loi de Fick)

## Solutions analytiques vs numériques

### Solutions exactes
- **Équations spéciales** : Bessel, Legendre, etc.
- **Fonctions spéciales** : solutions en série
- **Transformées intégrales** : Laplace, Fourier

### Simulation numérique
- **Méthodes aux différences finies**
- **Éléments finis** : discrétisation spatiale
- **Méthodes spectrales** : approximation par séries

## Théorie qualitative

### Points d'équilibre
Solutions constantes : f(y*) = 0

### Stabilité
- **Linéarisation** : analyse locale
- **Critères** : Routh-Hurwitz, Lyapunov

### Bifurcations
Changements qualitatifs lors de variation de paramètres

## Conclusion : Les EDO comme langage universel

Les équations différentielles offrent un cadre unificateur pour comprendre l'évolution des systèmes dynamiques. De la physique fondamentale à la biologie moléculaire, elles permettent de formuler des lois quantitatives précises et de faire des prédictions fiables.

Dans un monde en perpétuel changement, maîtriser les équations différentielles équivaut à posséder les clés pour déchiffrer les mécanismes cachés de la nature.

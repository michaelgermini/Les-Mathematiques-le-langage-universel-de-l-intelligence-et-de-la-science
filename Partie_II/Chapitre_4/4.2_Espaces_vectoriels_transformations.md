# 4.2. Espaces vectoriels et transformations

## Introduction : L'abstraction vectorielle

Au-delà des vecteurs concrets dans l'espace physique, l'algèbre linéaire généralise ces concepts à des espaces abstraits. Les espaces vectoriels et les transformations linéaires forment le cadre théorique qui permet d'étendre les idées géométriques à des contextes mathématiques plus généraux.

## Les espaces vectoriels : structures abstraites

### Définition axiomatique

Un espace vectoriel sur un corps K est un ensemble V muni de deux opérations :

#### Addition vectorielle
- **Fermeture** : ∀u,v ∈ V, u + v ∈ V
- **Associativité** : (u + v) + w = u + (v + w)
- **Commutativité** : u + v = v + u
- **Élément neutre** : ∃0 ∈ V, u + 0 = u
- **Inverse** : ∀u ∈ V, ∃-u ∈ V, u + (-u) = 0

#### Multiplication scalaire
- **Fermeture** : ∀λ ∈ K, ∀u ∈ V, λu ∈ V
- **Associativité** : λ(μu) = (λμ)u
- **Distributivité** : λ(u + v) = λu + λv et (λ + μ)u = λu + μu
- **Élément neutre** : 1u = u

### Exemples d'espaces vectoriels

#### Espaces classiques
- **ℝⁿ** : vecteurs colonnes à n composantes
- **ℂ** : nombres complexes (sur ℝ)
- **ℙₙ** : polynômes de degré ≤ n
- **F(X)** : fonctions continues sur un intervalle

#### Espaces plus abstraits
- **Espaces de suites** : ℓ², ℓ∞
- **Espaces de fonctions** : L²([a,b])
- **Espaces de matrices** : Mₙ(ℝ)

### Concepts fondamentaux

#### Sous-espaces vectoriels
- **Sous-ensemble fermé** : stable par addition et multiplication scalaire
- **Exemples** : droites, plans passant par l'origine
- **Intersection et somme** : opérations sur les sous-espaces

#### Générateurs et bases
- **Système générateur** : ensemble dont les combinaisons linéaires engendrent l'espace
- **Base** : système générateur minimal (linéairement indépendant)
- **Dimension** : cardinal d'une base

## Les transformations linéaires : morphismes vectoriels

### Définition et propriétés

Une transformation linéaire f : V → W est une application vérifiant :
- **Additivité** : f(u + v) = f(u) + f(v)
- **Homogénéité** : f(λu) = λf(u)

### Matrice associée

Toute transformation linéaire admet une matrice dans des bases fixées :
- **Matrice de f** : coefficients dans la base canonique
- **Composition** : (f ∘ g) correspond à M_f M_g
- **Inverse** : f bijective ↔ M_f inversible

### Noyau et image

#### Noyau (kernel)
- **Ker f** = {u ∈ V | f(u) = 0}
- **Sous-espace vectoriel** de V
- **Dimension** : liée au rang par théorème du rang

#### Image (range)
- **Im f** = {f(u) | u ∈ V}
- **Sous-espace vectoriel** de W
- **Rang** : dimension de l'image

### Théorème fondamental

Pour f : V → W, dim V = dim(Ker f) + dim(Im f)

## Applications linéaires importantes

### Projections et symétries

#### Projections orthogonales
- **Propriété** : P² = P
- **Géométrie** : projection sur un sous-espace
- **Applications** : traitement du signal, statistiques

#### Symétries
- **Propriété** : f² = Id
- **Exemples** : réflexions, rotations
- **Groupe orthogonal** : O(n)

### Isométries et similitudes

#### Isométries
- **Préservent les distances** : ||f(u) - f(v)|| = ||u - v||
- **Groupe des isométries** : translations, rotations, réflexions
- **Applications** : cristallographie, mécanique

#### Similitudes
- **Préservent les rapports** : distances multipliées par un facteur
- **Homothéties et similitudes directes**

## Diagonalisation et réduction

### Valeurs propres et vecteurs propres

#### Définition
Pour f : V → V, λ valeur propre si ∃u ≠ 0 tel que f(u) = λu
- **Spectre** : ensemble des valeurs propres
- **Multiplicité** : dimension de l'espace propre

#### Polynôme caractéristique
- **det(f - λId)** = 0
- **Racines** : valeurs propres
- **Théorème de Cayley-Hamilton** : f satisfait son polynôme caractéristique

### Diagonalisation

#### Cas diagonalisation
- **Matrice diagonalisable** : si elle admet n vecteurs propres linéairement indépendants
- **Base de vecteurs propres** : matrice diagonale dans cette base
- **Applications** : résolution d'équations différentielles

#### Théorème spectral
Pour les opérateurs auto-adjoints sur espaces de Hilbert :
- **Valeurs propres réelles**
- **Vecteurs propres orthogonaux**
- **Base hilbertienne**

## Espaces de Hilbert : structures avec produit scalaire

### Produit scalaire

Une forme bilinéaire symétrique définie positive :
- **⟨u,v⟩** hermitien si complexe
- **Norme** : ||u|| = √⟨u,u⟩
- **Angle** : cosθ = ⟨u,v⟩/(||u|| ||v||)

### Exemples d'espaces de Hilbert

- **ℓ²** : suites de carré sommable
- **L²([a,b])** : fonctions de carré intégrable
- **ℂⁿ** avec produit scalaire canonique

### Théorème de projection

Dans un espace de Hilbert, tout sous-espace fermé admet un projecteur orthogonal.

## Applications modernes

### Analyse fonctionnelle
- **Espaces de Sobolev** : pour les équations aux dérivées partielles
- **Théorie spectrale** : vibration des membranes, mécanique quantique

### Traitement du signal
- **Transformée de Fourier** : analyse fréquentielle
- **Filtres linéaires** : convolution
- **Compression** : ondelettes

### Apprentissage automatique
- **Séparation linéaire** : SVM (Support Vector Machines)
- **Analyse en composantes principales** : PCA
- **Réseaux neuronaux** : couches linéaires

## Conclusion : L'abstraction au service de l'universalité

Les espaces vectoriels et les transformations linéaires étendent les concepts géométriques intuitifs à des contextes mathématiques abstraits. Cette généralisation permet d'unifier des domaines apparemment disparates et de résoudre des problèmes complexes dans tous les secteurs de la science et de la technologie moderne.

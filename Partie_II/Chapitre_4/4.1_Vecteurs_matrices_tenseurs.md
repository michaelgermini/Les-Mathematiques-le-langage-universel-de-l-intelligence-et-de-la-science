# 4.1. Vecteurs, matrices, tenseurs

## Introduction : Les structures fondamentales de l'algèbre linéaire

L'algèbre linéaire constitue le langage mathématique des transformations et des espaces multidimensionnels. Au cœur de cette discipline se trouvent trois concepts fondamentaux : les vecteurs, les matrices et les tenseurs. Ces objets permettent de décrire et manipuler des données de dimension élevée avec une élégance remarquable.

Cette branche des mathématiques, développée au XIXe siècle par des mathématiciens comme Cayley, Sylvester et Grassmann, est aujourd'hui omniprésente : de la mécanique quantique aux réseaux de neurones, en passant par l'infographie et l'analyse de données.

---

## 1. Les vecteurs : flèches dans l'espace

### 1.1 Définition et propriétés

Un **vecteur** est un objet mathématique caractérisé par :
- **Magnitude** (ou norme) : sa longueur, notée ||v||
- **Direction** : son orientation dans l'espace
- **Sens** : positif ou négatif le long de cette direction

Formellement, un vecteur dans ℝⁿ est un n-uplet ordonné :

```
v = (v₁, v₂, ..., vₙ)
```

### 1.2 Opérations vectorielles fondamentales

#### Addition et multiplication scalaire
- **Addition** : v + w = (v₁ + w₁, v₂ + w₂, ..., vₙ + wₙ)
- **Multiplication scalaire** : λv = (λv₁, λv₂, ..., λvₙ)
- **Propriétés** : commutativité, associativité, distributivité

Ces opérations définissent la structure d'**espace vectoriel**.

#### Produit scalaire (ou produit intérieur)
- **Définition** : v · w = Σᵢ vᵢwᵢ = ||v|| ||w|| cos(θ)
- **Interprétation géométrique** : projection d'un vecteur sur l'autre
- **Orthogonalité** : v ⊥ w si et seulement si v · w = 0

#### Produit vectoriel (en dimension 3)
- **Définition** : v × w = (v₂w₃ - v₃w₂, v₃w₁ - v₁w₃, v₁w₂ - v₂w₁)
- **Propriétés** : anticommutativité (v × w = -w × v)
- **Interprétation** : vecteur perpendiculaire au plan (v, w), de norme égale à l'aire du parallélogramme

### 1.3 Norme et distance

La **norme euclidienne** d'un vecteur :
```
||v|| = √(v₁² + v₂² + ... + vₙ²)
```

Autres normes importantes :
- **Norme L¹** : ||v||₁ = |v₁| + |v₂| + ... + |vₙ| (distance de Manhattan)
- **Norme L∞** : ||v||∞ = max(|v₁|, |v₂|, ..., |vₙ|)
- **Norme Lᵖ** : ||v||ₚ = (Σᵢ |vᵢ|ᵖ)^(1/p)

## 2. Les matrices : tableaux de transformations

### 2.1 Définition et structure

Une **matrice** m×n est un tableau rectangulaire de nombres organisé en m lignes et n colonnes :

```
A = | a₁₁  a₁₂  ...  a₁ₙ |
    | a₂₁  a₂₂  ...  a₂ₙ |
    | ...  ...  ...  ... |
    | aₘ₁  aₘ₂  ...  aₘₙ |
```

L'élément à la ligne i et colonne j est noté aᵢⱼ ou A[i,j].

### 2.2 Opérations matricielles

#### Addition et multiplication scalaire
- **Addition** : (A + B)ᵢⱼ = Aᵢⱼ + Bᵢⱼ (matrices de même dimension)
- **Multiplication scalaire** : (λA)ᵢⱼ = λ · Aᵢⱼ

#### Multiplication matricielle
Pour A (m×p) et B (p×n), le produit AB est une matrice m×n :
```
(AB)ᵢⱼ = Σₖ Aᵢₖ · Bₖⱼ
```

**Propriétés cruciales** :
- **Non-commutativité** : AB ≠ BA en général
- **Associativité** : A(BC) = (AB)C
- **Distributivité** : A(B + C) = AB + AC

### 2.3 Opérations spéciales

#### Transposition
La **transposée** Aᵀ échange lignes et colonnes :
```
(Aᵀ)ᵢⱼ = Aⱼᵢ
```

#### Matrice inverse
Pour une matrice carrée A, son **inverse** A⁻¹ vérifie :
```
A · A⁻¹ = A⁻¹ · A = I (matrice identité)
```

Une matrice est **inversible** si et seulement si son déterminant est non nul.

#### Déterminant
Le **déterminant** det(A) mesure le facteur d'échelle de la transformation :
- det(A) = 0 : transformation non inversible (écrasement)
- |det(A)| = volume de l'image du cube unité
- signe de det(A) : préservation ou inversion de l'orientation

### 2.4 Matrices particulières

| Type | Propriété | Exemple d'usage |
|------|-----------|-----------------|
| **Identité** I | Iᵢⱼ = 1 si i=j, 0 sinon | Élément neutre |
| **Diagonale** | Aᵢⱼ = 0 si i≠j | Mise à l'échelle |
| **Symétrique** | A = Aᵀ | Covariance |
| **Orthogonale** | AᵀA = I | Rotations |
| **Hermitienne** | A = A† (conjuguée transposée) | Mécanique quantique |

## 3. Les tenseurs : généralisation multidimensionnelle

### 3.1 Définition tensorielle

Un **tenseur** d'ordre n est une généralisation des vecteurs (ordre 1) et matrices (ordre 2) à n indices :

| Ordre | Objet | Exemple |
|-------|-------|---------|
| 0 | Scalaire | Température |
| 1 | Vecteur | Vitesse |
| 2 | Matrice | Contraintes |
| 3 | Tenseur 3D | Élasticité |
| 4 | Tenseur 4D | Courbure de Riemann |

### 3.2 Tenseurs en physique

#### Tenseur des contraintes (mécanique des milieux continus)
Le tenseur σᵢⱼ décrit l'état de contrainte en un point :
- **Composantes diagonales** : contraintes normales
- **Composantes hors-diagonale** : contraintes de cisaillement
- **Équations de Cauchy** : équilibre mécanique

#### Tenseur métrique (relativité générale)
Le tenseur g_μν définit la géométrie de l'espace-temps :
```
ds² = g_μν dx^μ dx^ν
```
Les équations d'Einstein relient courbure et matière :
```
G_μν = 8πG/c⁴ · T_μν
```

### 3.3 Opérations tensorielles

#### Contraction
Réduction d'ordre par sommation sur indices répétés :
```
Trace(A) = Σᵢ Aᵢᵢ
```

#### Produit tensoriel (Kronecker)
Pour A (m×n) et B (p×q), le produit A ⊗ B est une matrice (mp×nq) :
```
A ⊗ B = | a₁₁B  a₁₂B  ...  a₁ₙB |
        | a₂₁B  a₂₂B  ...  a₂ₙB |
        | ...   ...   ...  ...  |
        | aₘ₁B  aₘ₂B  ...  aₘₙB |
```

## 4. Applications modernes

### 4.1 Traitement d'images

Une image couleur est un tenseur d'ordre 3 :
- **Dimensions** : hauteur × largeur × canaux (RGB)
- **Transformations** : rotation, mise à l'échelle (matrices 2×2)
- **Filtrage** : convolution avec noyaux matriciels

### 4.2 Mécanique quantique

- **Vecteurs d'état** : |ψ⟩ ∈ ℂⁿ (espace de Hilbert)
- **Observables** : matrices hermitiennes (A = A†)
- **Évolution** : équation de Schrödinger (matrice hamiltonienne)
- **Intrication** : produit tensoriel d'espaces

### 4.3 Apprentissage automatique

- **Données** : tenseurs multidimensionnels (batch × features × ...)
- **Couches denses** : transformations affines y = Wx + b
- **Convolutions** : tenseurs de filtres 4D
- **Attention** : produits matriciels Q·Kᵀ/√d

### 4.4 Graphiques 3D et jeux vidéo

- **Transformations** : matrices 4×4 (homogènes)
- **Rotation** : matrices orthogonales, quaternions
- **Projection** : perspective et orthographique
- **Shaders** : calculs vectoriels massivement parallèles

## 5. Implémentation numérique

### 5.1 Bibliothèques standards

```python
import numpy as np

# Vecteurs
v = np.array([1, 2, 3])
w = np.array([4, 5, 6])
dot_product = np.dot(v, w)  # Produit scalaire
cross_product = np.cross(v, w)  # Produit vectoriel

# Matrices
A = np.array([[1, 2], [3, 4]])
B = np.array([[5, 6], [7, 8]])
product = A @ B  # Multiplication matricielle
inverse = np.linalg.inv(A)  # Inverse
det = np.linalg.det(A)  # Déterminant

# Tenseurs (exemple : image RGB)
image = np.random.rand(256, 256, 3)  # Tenseur 3D
```

### 5.2 Considérations de performance

- **Vectorisation** : éviter les boucles Python explicites
- **Broadcasting** : opérations sur tenseurs de dimensions différentes
- **GPU** : PyTorch, TensorFlow pour calcul parallèle massif
- **Précision** : float32 vs float64 selon les besoins

## Conclusion : L'abstraction au service de l'application

Vecteurs, matrices et tenseurs forment le langage universel pour décrire les transformations linéaires et les structures multidimensionnelles. Leur puissance réside dans leur capacité à :

1. **Généraliser** des concepts intuitifs à des espaces de dimension arbitraire
2. **Unifier** des domaines apparemment distincts (physique, informatique, statistiques)
3. **Permettre** des calculs efficaces grâce à des algorithmes optimisés

De la compression d'images aux équations de la relativité, ces structures mathématiques sont les briques fondamentales de la science moderne.

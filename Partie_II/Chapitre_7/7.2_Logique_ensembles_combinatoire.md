# 7.2. Logique, ensembles et combinatoire

## Introduction

Ces trois piliers des mathématiques discrètes forment le socle de l'informatique théorique et de la pensée formelle. La logique structure le raisonnement, la théorie des ensembles fournit le langage fondamental des mathématiques, et la combinatoire compte et organise les configurations possibles.

---

## 1. Logique mathématique

### 1.1 Logique propositionnelle

La logique propositionnelle manipule des propositions (vraies ou fausses) via des connecteurs.

#### Connecteurs logiques

| Symbole | Nom | Signification | Table de vérité |
|---------|-----|---------------|-----------------|
| ¬ | Négation | NON | ¬V=F, ¬F=V |
| ∧ | Conjonction | ET | V∧V=V, sinon F |
| ∨ | Disjonction | OU | F∨F=F, sinon V |
| → | Implication | SI...ALORS | F→X=V, V→F=F |
| ↔ | Équivalence | SI ET SEULEMENT SI | mêmes valeurs |
| ⊕ | OU exclusif | XOR | différentes valeurs |

#### Lois fondamentales

**Lois de De Morgan** :
```
¬(A ∧ B) ≡ ¬A ∨ ¬B
¬(A ∨ B) ≡ ¬A ∧ ¬B
```

**Distributivité** :
```
A ∧ (B ∨ C) ≡ (A ∧ B) ∨ (A ∧ C)
A ∨ (B ∧ C) ≡ (A ∨ B) ∧ (A ∨ C)
```

**Absorption** :
```
A ∧ (A ∨ B) ≡ A
A ∨ (A ∧ B) ≡ A
```

#### Formes normales
- **FNC** (Forme Normale Conjonctive) : conjonction de clauses (disjonctions)
- **FND** (Forme Normale Disjonctive) : disjonction de termes (conjonctions)
- Toute formule peut être mise sous ces formes

### 1.2 Logique des prédicats

Extension de la logique propositionnelle avec variables et quantificateurs.

#### Quantificateurs
- **Universel** ∀ : "pour tout" — ∀x P(x) est vrai si P(x) est vrai pour tout x
- **Existentiel** ∃ : "il existe" — ∃x P(x) est vrai si P(x) est vrai pour au moins un x

#### Négation des quantificateurs
```
¬(∀x P(x)) ≡ ∃x ¬P(x)
¬(∃x P(x)) ≡ ∀x ¬P(x)
```

#### Exemple
"Tout nombre pair supérieur à 2 est la somme de deux nombres premiers" (conjecture de Goldbach) :
```
∀n (pair(n) ∧ n > 2 → ∃p ∃q (premier(p) ∧ premier(q) ∧ n = p + q))
```

### 1.3 Applications de la logique

#### Circuits logiques
- **Portes** : AND, OR, NOT, NAND, NOR, XOR
- **Algèbre de Boole** : base de la conception des processeurs
- **Minimisation** : cartes de Karnaugh, algorithme de Quine-McCluskey

#### Vérification formelle
- **Model checking** : vérification exhaustive des états d'un système
- **Preuve de programmes** : logique de Hoare, invariants de boucle
- **Spécification** : logique temporelle (LTL, CTL)

#### Intelligence artificielle symbolique
- **Systèmes experts** : règles SI-ALORS
- **Planification** : STRIPS, PDDL
- **Bases de connaissances** : logiques de description (OWL)

## 2. Théorie des ensembles

### 2.1 Notions de base

Un **ensemble** est une collection d'objets distincts appelés éléments.

#### Notation
- **Extension** : A = {1, 2, 3}
- **Compréhension** : A = {x ∈ ℕ | x < 4}
- **Appartenance** : x ∈ A (x est élément de A)
- **Inclusion** : A ⊆ B (tout élément de A est dans B)

#### Ensembles remarquables
- **∅** : ensemble vide
- **ℕ** : entiers naturels {0, 1, 2, ...}
- **ℤ** : entiers relatifs {..., -1, 0, 1, ...}
- **ℚ** : rationnels
- **ℝ** : réels
- **ℂ** : complexes

### 2.2 Opérations sur les ensembles

| Opération | Notation | Définition |
|-----------|----------|------------|
| **Union** | A ∪ B | {x | x ∈ A ou x ∈ B} |
| **Intersection** | A ∩ B | {x | x ∈ A et x ∈ B} |
| **Différence** | A \ B | {x | x ∈ A et x ∉ B} |
| **Différence symétrique** | A △ B | (A \ B) ∪ (B \ A) |
| **Complémentaire** | Aᶜ | Ω \ A (dans un univers Ω) |
| **Produit cartésien** | A × B | {(a,b) | a ∈ A, b ∈ B} |
| **Ensemble des parties** | P(A) | {X | X ⊆ A} |

#### Propriétés algébriques
- **Commutativité** : A ∪ B = B ∪ A, A ∩ B = B ∩ A
- **Associativité** : (A ∪ B) ∪ C = A ∪ (B ∪ C)
- **Distributivité** : A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)
- **Lois de De Morgan** : (A ∪ B)ᶜ = Aᶜ ∩ Bᶜ

### 2.3 Cardinalité et infini

#### Ensembles finis
- |A| = nombre d'éléments
- |P(A)| = 2^|A|

#### Ensembles infinis dénombrables
- **Définition** : peut être mis en bijection avec ℕ
- **Exemples** : ℕ, ℤ, ℚ
- **Cardinal** : ℵ₀ (aleph-zéro)

#### Ensembles non dénombrables
- **Théorème de Cantor** : ℝ n'est pas dénombrable (argument diagonal)
- **Cardinal du continu** : 𝔠 = 2^ℵ₀
- **Hypothèse du continu** : il n'existe pas de cardinal entre ℵ₀ et 𝔠

#### Hiérarchie des infinis
```
|ℕ| = ℵ₀ < |ℝ| = 2^ℵ₀ < |P(ℝ)| = 2^(2^ℵ₀) < ...
```
Le théorème de Cantor : |P(A)| > |A| pour tout ensemble A.

### 2.4 Axiomatique ZFC

La théorie des ensembles de Zermelo-Fraenkel avec l'axiome du choix (ZFC) :

1. **Extensionnalité** : deux ensembles sont égaux ssi ils ont les mêmes éléments
2. **Paire** : pour tout a, b, il existe {a, b}
3. **Union** : pour tout ensemble d'ensembles, leur union existe
4. **Parties** : P(A) existe pour tout A
5. **Infini** : il existe un ensemble infini
6. **Remplacement** : l'image d'un ensemble par une fonction est un ensemble
7. **Fondation** : pas de chaîne descendante infinie d'appartenance
8. **Choix** : pour toute famille d'ensembles non vides, on peut choisir un élément dans chacun

## 3. Combinatoire

### 3.1 Dénombrement de base

#### Principe multiplicatif
Si une tâche se décompose en k étapes avec n₁, n₂, ..., nₖ choix :
```
Nombre total = n₁ × n₂ × ... × nₖ
```

#### Principe additif
Si une tâche peut être réalisée de k manières mutuellement exclusives :
```
Nombre total = n₁ + n₂ + ... + nₖ
```

### 3.2 Arrangements et permutations

#### Arrangements sans répétition
Choisir k éléments parmi n avec ordre, sans répétition :
```
A(n,k) = n!/(n-k)! = n × (n-1) × ... × (n-k+1)
```

#### Arrangements avec répétition
Choisir k éléments parmi n avec ordre, avec répétition :
```
n^k
```

#### Permutations
Arrangements de n éléments parmi n :
```
P(n) = n!
```

#### Permutations avec répétitions
Permutations de n objets dont n₁ identiques, n₂ identiques, ... :
```
n! / (n₁! × n₂! × ... × nₖ!)
```

### 3.3 Combinaisons

#### Combinaisons sans répétition
Choisir k éléments parmi n sans ordre, sans répétition :
```
C(n,k) = (n k) = n! / (k!(n-k)!)
```

**Propriétés** :
- Symétrie : C(n,k) = C(n, n-k)
- Pascal : C(n,k) = C(n-1, k-1) + C(n-1, k)
- Somme : Σₖ C(n,k) = 2ⁿ

#### Combinaisons avec répétition
Choisir k éléments parmi n sans ordre, avec répétition :
```
C(n+k-1, k) = (n+k-1)! / (k!(n-1)!)
```

### 3.4 Formules avancées

#### Binôme de Newton
```
(a + b)ⁿ = Σₖ₌₀ⁿ C(n,k) aᵏ bⁿ⁻ᵏ
```

#### Formule d'inclusion-exclusion
```
|A₁ ∪ A₂ ∪ ... ∪ Aₙ| = Σᵢ |Aᵢ| - Σᵢ<ⱼ |Aᵢ ∩ Aⱼ| + Σᵢ<ⱼ<ₖ |Aᵢ ∩ Aⱼ ∩ Aₖ| - ...
```

#### Principe des tiroirs (pigeonhole)
Si n+1 objets sont répartis dans n tiroirs, au moins un tiroir contient ≥ 2 objets.

**Généralisation** : si kn+1 objets dans n tiroirs, au moins un tiroir contient ≥ k+1 objets.

#### Nombres de Catalan
```
Cₙ = C(2n,n)/(n+1)
```
Comptent : parenthésages, arbres binaires, chemins de Dyck, triangulations...

#### Nombres de Stirling
- **Première espèce** s(n,k) : permutations avec k cycles
- **Seconde espèce** S(n,k) : partitions de n éléments en k blocs non vides

### 3.5 Fonctions génératrices

#### Série génératrice ordinaire
```
F(x) = Σₙ aₙ xⁿ
```
Encode la suite (aₙ) ; les opérations sur F correspondent à des opérations sur la suite.

#### Exemples
- Suite constante (1,1,1,...) : 1/(1-x)
- Fibonacci : x/(1-x-x²)
- Factorielles : pas de forme close simple

## 4. Applications en informatique

### 4.1 Analyse d'algorithmes
- **Complexité** : compter les opérations (récurrences, arbres de récursion)
- **Structures de données** : arbres, tas, tables de hachage

### 4.2 Cryptographie
- **Comptage** : nombre de clés, espace des messages
- **Problèmes difficiles** : factorisation, logarithme discret

### 4.3 Théorie des codes
- **Codes correcteurs** : combinatoire des mots de code
- **Bornes** : Hamming, Singleton, Gilbert-Varshamov

### 4.4 Probabilités discrètes
- **Espaces d'échantillonnage** : ensembles d'événements
- **Dénombrement** : calcul de probabilités par comptage

---

## Conclusion

Logique, ensembles et combinatoire constituent le triumvirat des mathématiques discrètes :

1. **La logique** formalise le raisonnement et fonde la vérification
2. **Les ensembles** fournissent le langage universel des mathématiques
3. **La combinatoire** compte et organise les possibilités

Ces outils sont indispensables en informatique théorique, en cryptographie, en analyse d'algorithmes et partout où l'on manipule des structures discrètes.
